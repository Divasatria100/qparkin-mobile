## Key Takeaways

1. **Badge Flow**: Automatic - triggered by balance changes, cleared when page opened
2. **Notifications**: Manual - must be called explicitly in transaction handlers
3. **State Persistence**: Automatic - handled by NotificationProvider
4. **Navigation**: Consistent - all notifications navigate to `/point` route
5. **Error Handling**: Graceful - shows user-friendly messages for failures
